# 🧊 Smart Cold Chain Monitoring System

## 📖 Overview
The Smart Cold Chain Monitoring System is an IoT-based project designed to monitor the temperature of medicines during transportation. It uses ESP32, RFID, GPS, DHT11, and Email Alerts to ensure medicines are transported safely.

## ✨ Features
- RFID Authentication
- Medicine Selection
- Real-time Temperature Monitoring
- GPS Location Tracking
- Email Alerts on Temperature Breach
- Journey Start & End Tracking
- Compliance Report Generation

## 🛠 Components Used
- ESP32
- RFID RC522
- DHT11 Sensor
- GPS NEO-6M
- LCD I2C
- Keypad
- Wi-Fi

## 💻 Software Used
- Arduino IDE
- ESP32 Board Package
- ESP Mail Client Library
